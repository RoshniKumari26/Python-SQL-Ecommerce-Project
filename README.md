# 🛒 Python + SQL E-Commerce Sales Analysis

## 📌 Project Overview

This project focuses on analyzing an E-Commerce dataset using **Python** and **SQL** to uncover meaningful business insights. The analysis involves data cleaning, preprocessing, exploratory data analysis (EDA), SQL queries, and data visualization to understand customer behavior, sales performance, product trends, and order patterns.

The objective of this project is to demonstrate practical data analytics skills by solving real-world business problems using data.

---

## 🎯 Objectives

- Analyze customer purchasing behavior
- Identify sales trends and revenue patterns
- Perform SQL-based business analysis
- Clean and preprocess real-world data
- Visualize insights using Python libraries
- Generate actionable business recommendations

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/RoshniKumari26/Python-SQL-Ecommerce-Project.git
```

2. Navigate to the project directory

```bash
cd Python-SQL-Ecommerce-Project
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Open Jupyter Notebook

```bash
jupyter notebook
```

5. Run **Python+SQL(Ecommerce).ipynb**
